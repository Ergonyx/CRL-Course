Day 1 - Percentages and Heights

Leave width and height alone! They're responsive by default! If you MUST change them, use percentages!

The DEFAULT width of a block level element (div, p, h1, etc) is always 100% -- That's 100 PERCENT! NOTE: Percentages are based on the parent element. For instance, this paragraph has a width of 75% of the blue box that it is in. The blue box has a width of 80% of the overall screen space.

Day 2 - Getting Familiar With Relative Units

em and rem are relative units of measurement based on the font-size of either the parent or root (html) element.

em changes based on the font-size of the parent element (or reverts to the default of 16px. em also compounds as you go. Example below.

font-size: 1.1em; is 16 * 1.1 = 17.6px.
font-size: 1.1em; is 16 * 1.1 (P) * 1.1 (C) = 19.36px.

font-size: 1.1em; is 16 * 1.1 (P) * 1.1 (C) * 1.1 (this) = 21.296px.

You can calculate the em equivalent for any pixel value with the formula em = desired element pixel value / parent element font-size in pixels.

rem values were invented to sidestep the compounding problem of em units. rem takes its size from the root (html) element allowing for a standard unit of measurement compared to the calculated em units.
