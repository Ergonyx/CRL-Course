Day 1 - Percentages and Heights

Leave width and height alone! They're responsive by default! If you MUST change them, use percentages!

The DEFAULT width of a block level element (div, p, h1, etc) is always 100% -- That's 100 PERCENT! NOTE: Percentages are based on the parent element. For instance, this paragraph has a width of 75% of the blue box that it is in. The blue box has a width of 80% of the overall screen space.

CSS IS AWESOME

width: 900px; will not get any smaller (or larger) than that 900 pixels. This paragraph is a prime example of what could go wrong. The same applies when setting a height except the text will overflow below the element.

By setting the width to 900px the previous line will cause devices with a width of less than 900 pixels to have a horizontal scrollbar.

If you need to give something a height, don't. -- Kevin Powell
